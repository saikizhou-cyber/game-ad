# 不同创意形式的制作管线

## 通用前期包

进入具体路线前锁定：平台和交付规格、核心操作及输赢状态、钩子与回报、视觉圣经、声音曲线、镜头分工、跨媒介交接帧、版本命名和目录结构。

## A. 纯实机／AE动效广告

适合核心循环本身就能阻止用户划走的产品。

1. 还原真实玩法规则和状态机。
2. 把首帧设计成难题、临界失败或强迫症状态，而不是中性棋盘。
3. 设计棋盘、输入、托盘/库存、进度、失败和成功状态。
4. 逐次展示可归因于玩家的点击或拖动。
5. 通过声音与微动效让状态变化有触感。
6. 中段必须出现新的紧张、发现或连击，不能等间隔重复点击。
7. 玩法可读性锁定后再加入Logo和CTA。

适用于益智、整理、Tile、合成、放置成长、火力成长和改造类产品。

## B. UGC／达人原生广告

适合同类用户行为和人格信任能够促进下载的产品。

1. 文案写成人的真实反应，而不是企业旁白。
2. 足够早地展示产品，证明口播不是空口承诺。
3. 跳切、字幕和构图符合目标平台的原生观感。
4. 为角色设计具体行为：尝试、犯错、惊讶、纠正、推荐。
5. 准备干净的实机插入镜头和字幕安全版。

避免长时间只说不展示、手机动作与录屏不匹配以及破坏UGC真实感的过度棚拍。

## C. 真人或AI短剧

只有角色问题能够自然产生“为什么要玩”时才使用。

1. 用画面建立目标、阻碍和倒计时。
2. 角色的身体动作必须物理可行。
3. 用桥接物、运镜、匹配剪辑、扫描、擦除、传送或屏幕转场进入产品。
4. 游戏动作必须解决或重新解释剧情问题。
5. 当角色回归能够增强回报时，结尾回到角色完成情绪闭环。

AI视频注意：

- 减少同一时间发生的复杂手部、道具和表情动作；
- 用参考帧锁定人物、道具、空间与光线；
- 把难动作拆成镜头，不强求不可靠的一镜到底；
- 生成无精确文字的干净底片，后期添加文案；
- 每个镜头明确尾帧姿势和下一镜头入场姿势。

## D. AI视觉展示／隐喻

适合不可能变形或世界观本身就是钩子的产品。

1. 每个视觉隐喻必须对应真实产品特性。
2. 用少量高质量关键帧锁定美术方向。
3. 通过可控运镜、材质变化和物体层级完成动画。
4. 在观众误以为是无关短片前加入确定性的UI或实机证明。
5. 结尾必须清楚识别产品。

## E. 2D／3D产品展示

适合视觉质感、物体行为或成长过程是主要卖点的产品。

1. 先验证剪影和明暗层级，再做细节。
2. 使用清晰的因果动画语法。
3. 特写证明机制，大全景展示规模和结果。
4. 建立可复用绑定、材质、相机和特效预设。
5. 品牌和UI文字使用确定性后期工具。

## F. 混合制作

当单一媒介无法同时提供钩子和产品证明时使用。

为每个边界建立交接契约：

| 属性 | 前一个镜头 | 后一个镜头 |
|---|---|---|
| 画幅与构图 | 精确裁切和主体尺寸 | 匹配入场构图 |
| 透视 | 镜头角度、地平线 | 等价或有动机的变化 |
| 色彩 | 主色和亮度 | 延续强调色或有目的对比 |
| 运动 | 方向、速度、模糊 | 延续运动或冲击剪辑 |
| 物体 | 剪影和屏幕位置 | 同一锚点或合理变形 |
| 声音 | 预先进入、撞击或吸入 | 声音延续与新环境 |

先验证最困难的跨媒介边界。

## 声音管线

按层设计：

1. **钩子瞬态**：警报、撞击、吸气、断裂或突然安静；
2. **环境底层**：房间、机场、机器、风或设备底噪；
3. **输入拟音**：点击、拖动、拿起、落下、滑动；
4. **机制反馈**：锁定、解锁、匹配、连击、失败、进度；
5. **回报释放**：爆发、提示音、空间打开、角色反应；
6. **BGM曲线**：紧张、运动、提升、CTA句点。

不能通过单纯提高音量修补不清楚的画面逻辑。

## 版本策略

- V1/V2：更换钩子类型；
- A/B：同钩子下更换首个动作或文案；
- 节奏变量：快速证明与ASMR释放；
- 回报变量：角色闭环与玩法完成；
- CTA变量：挑战型、利益型或直接下载。

需要准确归因时，每个测试只更换一个主要变量。
